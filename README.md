# Auto Clicker for Windows

Простое приложение для автоматического кликания левой кнопкой мыши в течение заданного времени.

## 🚀 Для обычных пользователей

### Требования
- Windows 10 и выше
- Интернет-соединение только для скачивания

### Быстрый старт
1. [Скачать готовую версию](https://github.com/dluuhbiu/electron-auto-clicker/releases) (файл .exe)
2. Запустить скачанный файл
3. Нажать кнопку "Начать кликать"
4. Навести курсор на нужное место

### Как работает
- Нажмите "Начать" - приложение начнет кликать каждые 100 мс в течение 10 секунд
- Кнопка становится неактивной во время работы
- По истечении времени кликание останавливается
- Все логи отображаются в консоли приложения

### Безопасность
Приложение не требует прав администратора, не модифицирует системные файлы и не собирает никаких данных.

## 👨‍💻 Для разработчиков

### Требования
- Node.js 16+ [скачать здесь](https://nodejs.org/)
- npm (устанавливается вместе с Node.js)
- Git (опционально)

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm start
```

### Сборка исполняемого файла

Для Windows:
```bash
npm install --save-dev electron-builder
npm run build-win
```

Готовый файл будет в папке `dist/`

## Архитектура

- **Main Process**: main.js - управляет процессами PowerShell
- **Renderer Process**: index.html - пользовательский интерфейс
- **Коммуникация**: IPC между main и renderer процессами
