# Автокликер для Windows

Electron приложение для автоматического кликания левой кнопкой мыши в течение 10 секунд.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

## Использование

1. Запустите приложение
2. Нажмите кнопку "Начать кликать"
3. Приложение будет кликать левой кнопкой мыши в течение 10 секунд

## Особенности

- Графический интерфейс на Electron
- Автоматическое кликание в течение 10 секунд
- Работает только на Windows (использует PowerShell)
- Простота использования

## Сборка исполняемого файла

Для создания исполняемого файла .exe:

```bash
npm install --save-dev electron-builder
npm run build-win
```

Файл будет находиться в папке dist/
